## 📖 Overview

Voyago is a modern travel planning web application designed to provide users with an intuitive and visually engaging platform for planning their trips. The application features a clean, responsive interface with destination discovery, user authentication, favorites management, and trip organization.

The project focuses on delivering an elegant user experience while showcasing modern frontend development practices using **HTML5, CSS3, and Vanilla JavaScript**.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/aishwarya2401823209-collab/Voyago--AI-Travel-Planner.git
```

### Navigate to the Project

```bash
cd Voyago--AI-Travel-Planner
```

### Run the Application

Simply open **index.html** in your preferred web browser.

No additional setup or installation is required.
